Executive Summary

For the initial Adobe LLM Optimizer trial, Philippine Airlines should be configured as a single primary brand using https://philippineairlines.com/, with the brand monitored first in the Philippines (PH) market. While several category areas are valid for the brand, the trial should concentrate prompt coverage on just two priority categories: Booking & Destinations and Baggage & Travel Policies. This focused model provides stronger signal and clearer early insights than spreading the prompt budget too broadly.
