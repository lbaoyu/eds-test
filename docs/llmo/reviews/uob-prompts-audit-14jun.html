06 Adding prompts well

UOB's uploaded prompts are well-formed, including the dark ones — they are sound real-customer questions. The guidance here is therefore light: keep prompts true to their intent class, and add new ones the same way as the library grows.

Match the intent class

Branded prompts (service) and unbranded prompts (discovery) measure two different things, so each stays in its own class. A dark unbranded prompt is a signal to strengthen content for that topic — the two dark prompts are exactly the discovery questions UOB wants to win.

When a prompt genuinely needs work, refine within its intent class

If a prompt is vague or multi-intent, sharpen it for clarity and context — keeping it unbranded if it is a discovery question. Add a market, a persona or a use case.
