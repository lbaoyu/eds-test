Recommended "Other Brands" for Market Tracking

For HKTB, "Other Brands" should be configured by category / intent cluster, not as one flat list. The prompt set spans planning, food, stay, culture, events, and city comparisons, so the comparator set should reflect that structure.

Recommendation: use a focused set of travel platforms for planning and discovery prompts, and a separate destination-authority set for city-comparison prompts.

Category 2

Core

Setup Guidance

How to configure the list well

Guidance Recommendation
Keep the list focused Start with around 8–10 brands, not a long tail.
Use platform competitors where appropriate Use Tripadvisor, Klook, Booking.com, Agoda, OpenRice, and Time Out for planning, food, and stay prompts.
Use destination authorities for comparison prompts Use Visit Singapore, GoTokyo, Visit Seoul, and Amazing Thailand for city-vs-city prompts.
Keep entity types consistent Avoid mixing city entities and organization entities inconsistently in the same competitive logic.
Bottom Line

Recommended Market Tracking structure for HKTB

  • Planning / booking platforms: Tripadvisor, Klook, Booking.com, Agoda
  • Food / local discovery: OpenRice, Time Out
  • Destination authorities: Visit Singapore, GoTokyo, Visit Seoul

This structure should produce cleaner, more actionable Brand Presence comparisons than one undifferentiated competitor list.
