Overall Read

What the current usage pattern means

Snapshot summary

HKTB already has enough pilot data to show that LLMO works best when prompts are closely tied to trusted, official visitor guidance. The clearest early traction appears in questions around planning a Hong Kong trip, what first-time visitors should do, seasonal events, and culture-focused experiences.

At the same time, the current signal is still highly concentrated and early-stage. Only a subset of the broader prompt library is represented in the current results, and the strongest performance is coming from prompts that are more explicit, more decision-oriented, and more clearly aligned with HKTB's official role as a trusted source.

Because the customer is still in free trial and we have just uploaded refined prompts, this is the right moment to treat the current output as a baseline checkpoint. We have not yet done opportunity analysis or fix-related work, so the story today is less about optimization outcomes and more about identifying where the strongest early relevance already exists.

What is working now

Current strengths

Positive signals
  • Official / trusted guidance prompts perform best, especially when the wording clearly asks for recommendations from Discover Hong Kong or official tourism guidance.
  • Trip-planning intent is already showing traction, particularly for itinerary guidance, trusted planning resources, and first-time visitor support.
  • Seasonal and culture themes stand out despite relatively small prompt counts, which is encouraging for tourism-led discovery use cases.
  • Branded prompts are materially stronger than generic prompts, suggesting that HKTB has an advantage when the query is clearly connected to the official tourism board perspective.
Planning a Hong Kong Trip First-Time Visitor Questions Seasonal Travel & Events Culture & Heritage Experiences

In the current data, branded prompts average much stronger visibility than non-branded prompts, which reinforces the value of official-source framing.

What needs attention first

Highest-priority gaps

Priority gaps
  • The reporting sample is still narrow: the prompt library contains 200 configured prompts, but the current performance table only reflects 57 observed prompt results.
  • Most configured prompts are still marked as processing, so this is not yet a mature performance readout.
  • Coverage is uneven: a few themes are already generating useful signal, but broader tourism journeys still need time and validation.
  • Market alignment should be reviewed: most configured prompts currently show US as the market, which may not be the ideal long-term setup depending on HKTB's target visitor segments and localization strategy.
Most important near-term action: let the newly refined prompts run long enough to establish a more reliable baseline, then use that stabilized dataset for opportunity analysis and page / content prioritization.

Next focus

HKTB does not appear to have a demand problem so much as a signal distribution problem. A relatively broad pilot library already exists, but the current measurable wins are concentrated in prompts that are specific, official-source-oriented, and closely tied to real visitor planning intent.

This suggests a clear next phase: rather than simply adding more prompts immediately, the better path is to observe the refined prompt set, identify which prompt structures consistently earn mentions and owned citations, and then expand from those winning patterns into adjacent visitor journeys.

Client-facing summary

Final takeaway

Hong Kong Tourism Board has already established a credible early LLMO baseline during the free-trial phase. The strongest current signals are appearing in official trip planning, first-time visitor guidance, seasonal events, and culture / heritage discovery.

  • Keep observing the refined prompt set to build a stronger and more stable baseline.
  • Preserve and deepen the topics already showing traction, especially official planning and trusted visitor guidance themes.
  • Review market targeting before making broader strategic judgments.
  • Move into opportunity analysis next, once the updated prompt library has had time to generate a better evidence base.
  • Delay fix-related optimization work until after that analysis, so effort is directed toward the highest-value pages and journeys.

In short: HKTB is at the right pilot stage to validate where official tourism guidance can win in AI experiences, and the newly refined prompts should give the next snapshot a much stronger basis for action.
