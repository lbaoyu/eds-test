Overall interpretation

What the current usage pattern means

Key summary

HKTB's current LLMO setup is already producing real signal, but that signal is concentrated in a relatively small part of the overall prompt library. This is actually a positive sign: it shows that HKTB has already established workable and valuable prompt patterns in a small number of core Categories and Topics. The next step should be to make the overall program more precise, easier to scale, and more consistently tied to owned site content.

Based on current performance, the most effective prompts are usually the ones that most closely match the real questions travelers ask. They are more concrete, more planning-oriented, and more clearly tied to specific visitor needs. By contrast, the weaker parts of the library tend to be broader, more generic, or less consistently structured.

The current prompt volume is already enough to produce directional insight across a selected set of Topics; however, if HKTB wants a more complete view of the full website, a broader set of traveler needs, and a more complete travel journey, it will still need wider prompt coverage and a deeper observation design.

What is working now

Current strengths

Positive signals
  • High-intent prompts perform best, especially in Topics related to trip planning, official recommendations, events, and first-time visitor guidance.
  • Brand and authority-led prompts create the clearest visibility signal, showing that HKTB can build an advantage when traveler intent is explicit.
  • The current usage level is already enough to identify repeatable winning patterns and define a more focused optimization roadmap.
  • Trip-planning journeys are already a clear growth anchor, with Planning a Hong Kong Trip currently generating the highest mention volume.

What needs priority attention

Highest-priority gaps

Priority fix
  • Performance is highly concentrated: among the measured prompts, 36 out of 57 still generate 0% visibility.
  • Owned citation depth is still limited: only a small portion of measured prompts is currently producing owned citations.
  • Coverage is uneven: some Topics already show traction, but much of the configured library is not yet turning into meaningful observed signal.
  • AI accessibility is a major constraint: current opportunities point to crawl restrictions and content readability issues, which limit how effectively AI systems can discover and cite HKTB pages.
Most urgent action: first improve AI accessibility and page readability. This is the basic requirement for turning existing prompt strength into broader visibility, citation growth, and more consistent downstream traffic.
Usage quality

What the current prompt mix is telling us

Next focus

HKTB's current situation is not that it lacks prompts entirely; rather, prompt quality and prompt coverage are unevenly distributed. For a selected set of Topics, the existing prompt library is already sufficient to support an initial read. But if HKTB wants a more complete evaluation of how the wider site performs in AI search and AI answers, it will still need broader prompt coverage and more complete journey design.

This makes the next step very clear: move from simply adding more prompts toward refining the strongest prompts and gradually broadening coverage. The right sequence is to keep the Topics that are already proving effective, improve prompt phrasing and structure, remove low-signal or overly broad prompts, and then expand into more high-value traveler scenarios.

What HKTB should do next

Summary

Client-ready summary

Final takeaway

Hong Kong Tourism Board has already built a credible LLMO foundation and is already showing clear early results. The most direct opportunity now is to make the overall program more structured, more page-connected, and more strategically focused.

  • Keep and strengthen the prompts that are already working in planning, first-time visitor guidance, events, and culture.
  • Improve AI accessibility and readability so the current prompt program can convert more effectively into citations and traffic.
  • Expand gradually from winner Topics, moving from a small number of high-value Categories into broader site content and fuller traveler journeys.
  • Use a co-work pilot to turn the current early insight into faster, page-level and content-level optimization actions.

Once these adjustments are made, HKTB will have a stronger chance to turn its current early results into more stable, broader, and more scalable LLMO visibility outcomes.
