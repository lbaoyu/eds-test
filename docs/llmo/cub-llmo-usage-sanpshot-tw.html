目前使用情況代表什麼

重點摘要

國泰世華目前的 LLMO 設定已經產生真實訊號，但這些訊號主要集中在整體 prompt library 中的一小部分。這其實是正向訊號：代表國泰世華已經在部分核心 Categories 與 Topics 上，建立出可行且具價值的高品質 prompts；下一步應聚焦於讓整體計畫 更精準、更貼近在地市場，也更容易持續擴大。

從目前表現來看，最有效的 prompts 通常最接近真實客戶會提出的問題。它們更具體、決策意圖更明確，也更貼近清楚的產品或服務情境。相對之下，較弱的部分通常過於寬泛、泛用，或結構不夠一致。

目前的 prompts 數量，對於 少數已選定的 Categories 與 Topics，已足以產生方向性洞察；但若要更完整反映 整體網站、更多產品線，以及更完整的客戶旅程，仍需要更廣的 prompt 覆蓋與更深入的觀測設計。

目前 prompt 組合透露的訊號

下一步聚焦

國泰世華目前並不是完全缺少 prompts，而是 prompt 品質與覆蓋範圍分布不均。現有 prompt library 對於已選定的部分 Topics，已足以支持初步判讀；但若希望更完整評估整體網站在 AI 搜尋與回答中的表現，則仍需要更廣的 prompts 覆蓋與更完整的旅程設計。

這表示下一步非常明確：從 單純增加 prompts 轉向 精修核心 prompts 並逐步擴大覆蓋。優先順序應該是保留已證明有效的 Topics、優化 prompt 寫法與結構、移除過於寬泛或低訊號 prompts，接著再延伸到更多高價值的產品與服務情境。

建議行動

國泰世華下一步該怎麼做

行動方向
優先順序 建議事項 重要原因
1 將 market 從 US 改為 TW。 這是最重要的設定修正，可立即提升在地相關性。
2 補充 aliases。 aliases 可幫助 LLMO 更完整理解品牌、產品與服務的不同表述方式，提升 Topics 與 prompts 的涵蓋度與相關性。
3 圍繞高表現 Topics 精修核心 prompts。 國泰世華已經有明確的勝出 Topics，應先把這些優勢做深做強。
4 整理 prompt taxonomy 與 labeling。 更清楚的結構會讓報表、優先順序與後續優化更可靠。
5 移除或改寫過於寬泛、重複或低訊號的 prompts。 降低噪音後，整體計畫會更聚焦、更具可操作性。
6 在 opportunities 中嘗試使用 Optimise on Edge 功能。 對 trial users 而言，最多可先針對 10 個頁面進行測試，快速驗證頁面層級的優化機會與實際改善方向。
7 逐步擴展到更多產品頁、服務情境與完整網站旅程。 若要更完整反映整體網站表現，需要更廣的 prompt 覆蓋與更全面的觀測設計。

共創機會

建議啟動 co-work pilot

下一步建議

以目前試用階段的成果來看，國泰世華已經在部分核心 Categories 與 Topics 上看見明確訊號。這是一個很適合啟動 co-work pilot 的時間點：在既有成果基礎上，由雙方共同聚焦高價值主題，進一步加快 learning cycle，並把目前的方向性洞察轉化為更具體、可執行的優化策略。

為什麼現在適合進行 co-work pilot： 國泰世華已經有初步成果，但目前觀測仍集中在部分主題。透過短期、聚焦的共創 pilot，可以更快驗證哪些 Categories、Topics 與 prompts 最值得優先擴大，並逐步延伸到更多網站內容與客戶旅程。
co-work pilot 可共同完成的項目 預期價值
共同挑選高優先 Categories 與 Topics 聚焦最值得投入的主題，讓資源使用更集中、更有效率。
共同優化 prompts、aliases 與 market 設定 提升在地相關性與 entity 覆蓋，讓觀測結果更準確、更具參考價值。
共同擴展到更多產品頁與服務情境 把目前集中在少數主題的成果，延伸到更完整的網站覆蓋。
共同測試 opportunities 中的 Optimise on Edge 以少量頁面快速驗證頁面層級優化是否能帶來更明確的改善方向。
共同追蹤前後變化與 learnings 讓 pilot 不只是試驗，而是能形成明確 learnings 與後續決策依據。

對國泰世華而言，這樣的 co-work pilot 的價值在於：用相對低風險、低摩擦的方式，先驗證哪些擴大方向最值得投入。如果 pilot 顯示在更多 Categories、Topics 與 customer journeys 上都能穩定產出洞察，後續就更容易規劃更完整、可持續放大的 LLMO 策略。

建議可先以一小段期間、聚焦少數高價值主題進行 co-work pilot，再依成果逐步擴大範圍。

總結

客戶版本摘要

最終重點

國泰世華銀行已經建立具說服力的 LLMO 基礎，並且已出現清楚的早期成效。當前最直接的機會，在於讓整體計畫 更貼近台灣市場、更有結構，也更具策略聚焦。

  • 保留並強化 目前在卡片、海外使用、費用與 App 任務上已經有效的 prompts。
  • 修正 market 設定，並補充 aliases，讓整體計畫更完整反映台灣市場。
  • 由核心 Topics 逐步擴大，從少數高價值 Categories 延伸到更多網站內容與完整客戶旅程。
  • 結合 Optimise on Edge 與 co-work pilot，將目前的初步洞察更快轉化為頁面與內容層級的實際優化。

完成這些調整後，國泰世華將更有機會把目前的早期成效，轉化為更穩定、更可擴張的 LLMO 能見度成果。
