The pipeline

How each video was made

One connected flow, from a blank page to a video playing on a live AEM page. Each stage names the Adobe — or AI — tool that did the work.

01

Ideate & script

Started from the topic, not the tool — using AI to shape what each tutorial should cover and to draft the narration before anything was filmed.

FluffyJaws — Adobe internal AI Claude — script polish
02

Generate the avatar video

Turned the finished script into a presenter-led video with Firefly's text-to-avatar — no studio, no camera, on-brand and repeatable.

Adobe Firefly — text to avatar
03

Edit & finish

Quick trims and resizing in Express for fast format changes, then Premiere Pro for the cut, pacing and finishing touches.

Adobe Express — trim & resize Premiere Pro — edit
04

Encode for distribution

Rendered distributable versions in Media Encoder — the right codecs and outputs for delivery rather than the editing timeline.

Adobe Media Encoder
05

Manage & deliver

Stored and governed in AEM Assets, published through Dynamic Media on the Open API path, and served via Edge Delivery Services for fast, optimised playback.

AEM Assets — DAM Dynamic Media — Open API AEM Edge Delivery Services
06

Build the learning surface

Wrapped the videos in a clean, self-contained gallery page so the series is easy to browse and share — designed and built with AI assistance.

Claude — HTML & web design
