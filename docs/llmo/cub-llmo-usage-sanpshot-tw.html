整體判讀

目前使用情況代表什麼

重點摘要

國泰世華目前的 LLMO 設定已經產生真實訊號，但這些訊號主要集中在整體 prompt library 中的一小部分。這其實是正向訊號：代表國泰世華已經在部分核心 Categories 與 Topics 上，建立出可行且具價值的高品質 prompts；下一步應聚焦於讓整體計畫 更精準、更貼近在地市場，也更容易持續擴大。

從目前表現來看，最有效的 prompts 通常最接近真實客戶會提出的問題。它們更具體、決策意圖更明確，也更貼近清楚的產品或服務情境。相對之下，較弱的部分通常過於寬泛、泛用，或結構不夠一致。

目前的 prompts 數量，對於 少數已選定的 Categories 與 Topics，已足以產生方向性洞察；但若要更完整反映 整體網站、更多產品線，以及更完整的客戶旅程，仍需要更廣的 prompt 覆蓋與更深入的觀測設計。

需要優先處理的部分

最高優先缺口

優先修正
  • market 設定錯位：目前 prompt library 使用的是 US，但應改為 TW。
  • 表現高度集中：已完成的 prompts 中，只有部分 prompts 真正產生有意義的 mentions。
  • prompt 結構不夠一致：taxonomy 與 labeling 仍有優化空間，影響後續報告與管理品質。
  • 覆蓋面不均：卡片主題相對完整，但其他產品與網站內容的覆蓋仍有限。
最緊急的行動：先將 market 從 US 改為 TW。這是提升相關性、在地語言對齊與競品語境準確度的基礎。

目前 prompt 組合透露的訊號

下一步聚焦

國泰世華目前並不是完全缺少 prompts，而是 prompt 品質與覆蓋範圍分布不均。現有 prompt library 對於已選定的部分 Topics，已足以支持初步判讀；但若希望更完整評估整體網站在 AI 搜尋與回答中的表現，則仍需要更廣的 prompts 覆蓋與更完整的旅程設計。

這表示下一步非常明確：從 單純增加 prompts 轉向 精修核心 prompts 並逐步擴大覆蓋。優先順序應該是保留已證明有效的 Topics、優化 prompt 寫法與結構、移除過於寬泛或低訊號 prompts，接著再延伸到更多高價值的產品與服務情境。

建議行動

國泰世華下一步該怎麼做

行動方向
優先順序 建議事項 重要原因
1 將 market 從 US 改為 TW。 這是最重要的設定修正，可立即提升在地相關性。
2 圍繞高表現 Topics 精修核心 prompts。 國泰世華已經有明確的勝出 Topics，應先把這些優勢做深做強。

從試用走向完整導入的機會

成長機會

以目前試用階段的配置來看，國泰世華已經能在少數核心 Categories 中看見明確的方向與價值。若下一步希望更完整掌握整體網站在 AI 回答中的品牌能見度、內容覆蓋與客戶旅程表現，則更廣的 prompt 部署將能提供更全面的管理視角。

總結

客戶版本摘要

最終重點

國泰世華銀行已經建立具說服力的 LLMO 基礎，並且已出現清楚的早期成效。當前最直接的機會，在於讓整體計畫 更貼近台灣市場、更有結構，也更具策略聚焦。

  • 保留並強化 目前在卡片、海外使用、費用與 App 任務上已經有效的 prompts。
  • 修正 market 設定，讓整體計畫反映台灣，而不是美國。
  • 由核心 Topics 逐步擴大，從少數高價值 Categories 延伸到更多網站內容與完整客戶旅程。

完成這些調整後，國泰世華將更有機會把目前的早期成效，轉化為更穩定、更可擴張的 LLMO 能見度成果。
