06 Adding prompts well

KBank's uploaded prompts are well-formed, including the dark ones — they are sound real-customer questions. The guidance here is therefore light: keep prompts true to their intent class, and add new ones the same way as the library grows.

Match the intent class

Branded prompts (card questions naming KBank) and unbranded prompts (fund and card discovery) measure two different things, so each stays in its own class. A dark unbranded prompt is a signal to strengthen content for that topic — the 16 dark investment prompts are exactly the discovery questions KBank wants to win.

When a prompt genuinely needs work, refine within its intent class

If a prompt is vague or multi-intent, sharpen it for clarity and context — keeping it unbranded if it is a discovery question. Add a goal, a risk level or a horizon.
