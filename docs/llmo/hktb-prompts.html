Revision Logic

How the wording was adjusted

Source-seeking, not over-branded
  • Removed unnatural brand forcing such as direct "Discover Hong Kong" phrasing where a real user would more naturally ask for trusted guidance.
  • Kept important SEO / event / multilingual prompts intact where exact phrasing mattered.
  • Shifted planning prompts toward trust and utility with wording like "Where can I find trusted information…" or "What should I know before visiting…"
  • Preserved the topic structure so the revised prompts still fit the existing 200-prompt configuration.
