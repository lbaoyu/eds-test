整體判讀

目前使用情況代表什麼

重點摘要

國泰世華目前的 LLMO 設定已經產生真實訊號，但這些訊號主要集中在整體 prompt library 中相對較小的一部分。這其實是正向訊號：代表國泰世華已經擁有一組可行、且具價值的高品質 prompts；下一步應該聚焦於讓整體計畫 更精準、更貼近在地市場，也更容易持續擴大。

從目前表現來看，最有效的 prompts 通常最接近真實客戶會提出的問題。它們更具體、決策意圖更明確，也更貼近清楚的產品或服務情境。相對之下，較弱的部分通常過於寬泛、泛用，或結構不夠一致。

目前有效的部分

現階段優勢

正向訊號
  • 高意圖的 prompts 表現最佳，特別是與卡片選擇、卡片使用、費用與 App 操作相關的主題。
  • 品牌與服務導向的 prompts 帶來最清楚的可見度訊號，顯示只要客戶意圖明確，國泰世華就有機會建立優勢。
  • 目前的使用量已足以找出可重複成功的模式，並建立更聚焦的優化路線圖。
  • 卡片相關旅程已是清楚的成長支點，因為這些主題已經顯示出明顯 traction。

需要優先處理的部分

最高優先缺口

優先修正
  • 市場設定錯位：目前 library 使用的是 US，但應改為 TW。
  • 表現高度集中：已完成 prompts 中，只有部分 prompts 真正產生有意義的 mentions。
  • Prompt 結構不夠一致：分類法與標籤仍有優化空間，影響後續報告與管理品質。
  • 覆蓋面不均：卡片主題相對完整，但相鄰的銀行旅程覆蓋較薄弱。
最緊急的行動：先將 market 從 US 改為 TW。這是提升相關性、在地語言對齊與競品語境準確度的基礎。
使用品質

目前 prompt 組合透露的訊號

下一步聚焦

國泰世華目前並不是 prompt 數量不足，而是 prompt 品質分布不均。整體 library 規模已足夠，但最強的結果來自較小、結構更好的 subset，而其餘 prompts 帶來的有效訊號相對有限。

這表示下一步非常明確：從 prompt 擴張 轉向 prompt 精修。優先順序應該是保留已證明有效的主題、優化 prompt wording 與結構、移除過於寬泛或低訊號 prompts，接著再逐步擴展到下一批高價值銀行旅程。

品牌／服務導向 prompts 明顯優於泛用 prompts 越貼近具體客戶意圖，越容易形成高品質的衡量結果。
卡片與 App 旅程目前表現領先 這些主題應持續作為下一階段精修與擴張的核心。

建議行動

國泰世華下一步該怎麼做

行動方向
優先順序 建議事項 重要原因
1 將 market 從 US 改為 TW。 這是最重要的設定修正，可立即提升在地相關性。
2 圍繞高表現主題精修核心 prompt set。 國泰世華已經有明確的勝出主題，應先把這些優勢做深做強。
3 整理 prompt taxonomy 與 labeling。 更清楚的結構會讓報表、優先順序與後續優化更可靠。
4 移除或改寫過於寬泛、重複或低訊號的 prompts。 降低噪音後，整體計畫會更聚焦、更具可操作性。
5 有節奏地擴展到相鄰銀行旅程。 下一波擴張應建立在既有優勢上，而不是平均式擴張。

總結

客戶版本摘要

最終重點

國泰世華銀行已經建立具說服力的 LLMO 基礎，並且已出現清楚的早期成效。當前最直接的機會，在於讓整體計畫 更貼近台灣市場、更有結構，也更具策略聚焦。

  • 保留並強化 目前在卡片、海外使用、費用與 App 任務上已經有效的 prompts。
  • 修正市場設定，讓整體計畫反映台灣，而不是美國。
  • 從量轉向質，以精修核心 prompt set、降低噪音為優先。

完成這些調整後，國泰世華將更有機會把目前的早期成效，轉化為更穩定、更可擴張的 LLMO 能見度成果。
